## 🚀 **Getting Started**

### **Quick Setup**
```bash
# Clone SnikDis Crypto
git clone <repository-url>
cd SnikDis-Crypto

# Install dependencies
npm install

# Configure your environment
cp env.template .env
# Add your API keys

# Start the platform
npm run dev
```
